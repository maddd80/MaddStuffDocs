# MaddStuff Docs

Documentation for MaddStuff projects.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run local development server:
   ```bash
   npm run docs:dev
   ```

3. Build for production:
   ```bash
   npm run docs:build
   ```

4. Preview production build:
   ```bash
   npm run docs:preview
   ```
