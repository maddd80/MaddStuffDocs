## Key Features

- **🎲 CS2-Style Opening**: Smooth, suspenseful spinning animation with deceleration and sound effects.
- **💎 Rarity System**: Configurable rarities from Common to Legendary with distinct visual styles.
- **🔄 NPC Traders**: Set up traders for players to exchange gathered resources for Gold or Items.
- **💰 Versatile Rewards**: Supports Items, Weapons, and Currency (Cash/Gold) as rewards.
- **📊 Weighted Drops**: Fine-tune your economy with precise drop rate percentages.
- **📜 Logging**: Detailed Discord webhook logs for transparency and security.

## Requirements

- [jo_libs](https://docs.jumpon-studios.com/jo_libs/)
- [madd_ui](https://docs.maddstuffs.com/scripts/ui/)
