## Key Features

- **📖 Persistent Details**: Create details that stay even after you leave the area.
- **🔍 Inspection System**: Other players can interact with details to read the full description.
- **🩹 Conditions**: Support for "Conditions" (e.g., injuries or temporary states) with separate timing.
- **⏳ Adjustable Duration**: Set how long a detail should last (up to 72 hours by default).
- **🛠️ Edit Mode**: Intuitive placement and time adjustment system.
