## Key Features

- **🏁 Dynamic Track Creation**: Easily create custom racing routes by placing checkpoints dynamically using controller or keyboard binds.
- **🗺️ Interactive Checkpoints**: Features adjustable checkpoint radius, 3D prop spawning at checkpoints, and GPS route rendering on the minimap.
- **⏱️ Competitive Races**: Supports countdowns, pre-race player freezing to prevent false starts, and real-time position updates.
- **🏆 Leaderboards**: Records track times and race outcomes into the database.
- **UI Integration**: Fully integrated with `madd_ui` for sleek notifications and menus.
