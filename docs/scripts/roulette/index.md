## Key Features

- **🎰 Realistic Roulette Rules**: Supports Straight, Split, Street, Corner, Column, Dozen, and more.
- **📍 Configurable Tables**: Add roulette tables to any location.
- **💰 Accurate Payouts**: Logic based on standard Las Vegas roulette payouts.
- **🎨 Modern UI**: Clean and intuitive interface for placing bets.
- **📜 Discord Logs**: Log every spin and payout for server security.
