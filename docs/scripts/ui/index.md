# Madd UI

Madd UI is the central user interface resource for the Madd's Stuffs script ecosystem. It provides high-performance, Vue 3-powered UI components and minigames that can be used by any developer to enhance their RedM server.

## Included Systems

- **🔔 Notifications**: Versatile notification system with multiple positions and durations.
- **🎮 Key Prompts**: RDR2-style inline key prompts for immersive interactions.
- **📊 Progress Bars**: Both time-based and skill-based (moving zone) progress bars.
- **🔐 Minigames**: Dial-style Safecracking and Skyrim-style Pin Lockpicking.
- **💬 Dialogue System**: A robust system for creating interactive NPC conversations with cinematic camera control.

## For Developers

Madd UI is designed to be an "all-in-one" UI provider. Many Madd resources require this script as a dependency to handle their visual output.
